# 大きい牌

ワンタップで300円を稼ぐ、バズり系置き配ゲーム。落とし前をつけられるか？

**Play:** https://keikamotushige-hub.github.io/okii-pai-puzzle/

## 遊び方

1. **置き配を実行する** — タップするだけ
2. **65%** → +300円（連続成功で倍率アップ）
3. **35%** → 脅迫イベント → **落とし前をつける**（50%で生還）
4. **レアイベント** — 伝説・奇跡・VIP心付けで大量獲得
5. ゲームオーバー後、称号と結果をポストして友達に挑戦させよう

## 称号

| 稼ぎ | 称号 |
|------|------|

| 600円〜 | 置き配係 |

| 3,000円〜 | 落とし前の達人 |
| 6,000円〜 | 300円の神 |
| 9,000円〜 | 大きい牌の王 |
