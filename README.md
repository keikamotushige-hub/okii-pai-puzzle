# 大きい牌 — 置き配パズル

架空の置き配パズルゲーム。正しい順番で麻雀牌を並べ、8ステージをクリアしよう。

**Play:** https://keikamotushige-hub.github.io/okii-pai-puzzle/

## 遊び方

1. **置き配パズル** — 注文どおりの順番で手牌をスロットに並べる
2. **落とし前** — 体力が尽きると神経衰弱ミニゲームで挽回チャンス
3. **8ステージクリア** — ステージが進むほど並べる牌の数が増える

## 技術

- 単一 HTML ファイル（ビルド不要）
- GitHub Pages でホスティング
